Components

MccTimerPicker

This is the main class of the component.

Inputs

Name Description
hideButtons: boolean Hide the buttons (confirm/cancel)
mccTimerPickerFormat: MccTimerPickerFormat Define the format that timer-picker will output the selected time (default: 12)
mccTimerPickerMin: string Define the minimum allowed time (default: 00:00 am)
mccTimerPickerMax: string Define the maximum allowed time (default: 12:00 pm)
btnCancel: string Cancel button text (default: Cancel)
btnConfirm: string Confirm button text (default: Confirm)

Outputs

Name Description
selected: string Output the selected timer

Directives

MccTimerPickerOriginDirective

This directive is used on the trigger component. If we associate the directive an input text, when the input is focused the timer picker will open on the screen

MccConnectedTimerPickerDirective

Directive used on the MccTimerPickerComponent to connect to an element ( MccTimerPickerOriginDirective)

Inputs

Name Description
mccConnectedTimerPickerOrigin: MccTimerPickerOriginDirective Element that timer picker has to connect
