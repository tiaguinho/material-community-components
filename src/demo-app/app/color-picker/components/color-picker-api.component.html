Components

MccColorPicker

This is the main class of the component.

Inputs

Name Description
disabled: boolean Disable color-picker pop up
usedColorLabel: string Change the label of the used colors collection
usedColorStart: string[] Set initial value for used colors collection
reverseUsedColors: boolean Reverse order of used colors list
hideHexForms: boolean Hide the hexadecimal color input form
hideEmpty: boolean Hide all empty slots of the default collection
hideTransparentUsedColors: boolean Hide transparent option of UsedColors
hideUsedColors: boolean Hide UsedColors default collection
hideButtons: boolean Hide the buttons (confirm/cancel)
hideColorPickerSelector: boolean Hide the color picker selector
colorPickerSelectorHeight: number Define new height of selector area
selectedColor: string Define initially selected color
isOpen: boolean Define if panel will initiate open
overlay: boolean Define if panel will be show inside overlay
usedSizeColors: number Set the size of used colors collection
btnCancelLabel: string Will replace the default text( Cancel) in cancel button
btnConfirmLabel: string Will replace the default text( Confirm) in confirm button

Outputs

Name Description
change: EventEmitter Event emitted when the color changed
selected: EventEmitter Event emitted when the color is selected
clickOut: EventEmitter Event emitted when is user clicks outside of the component
canceled: EventEmitter Event emitted when is user clicks on cancel button

MccColorPickerCollection

This class is used to pass your own collection of colors to be displayed inside the color picker.

Inputs

Name Description
hideEmpty: boolean Hide all empty slots inside de collection
label: string String with the name of the collection
colors: MccColorPickerOption[] Array of the colors that will be displayed inside the collection
size: number Limit of colors to be displayed
transparent: boolean Show transparent options (default is false)

Outputs

Name Description
changeColor Event that id emitted when a change ocurred

Services

MccColorPickerService

You can use service to add, get and reset the used colors component data.

Methods

Name Description
addColor(color: string): void Add new color to the used color collection
getColors(): Observable{{ '<' }}string[]{{ '>' }} Return observable of used color collection
resetUsedColors(): void Clear the used color collection

Directives

Using the MccColorPickerOrigin and MccConnectedColorPicker, you also can connect the color picker with any input, textarea or select element.

MccColorPickerOrigin

This directive goes in the element (input, textarea or select) that will be connected to the color picker.

Outputs

Name Description
changes: string Emit changes from the origin

MccConnectedColorPicker

This directive is used in the color picker that you want's to connect.

Inputs

Name Description
mccConnectedColorPickerOrigin: MccColorPickerOrigin Element that color picker has to connect

Interfaces & Types

MccColorPickerOption

Is a type created to handle colors. This type accept string or MccColorPickerItem.

MccColorPickerItem

This interface is used when you have the name of the color. This can help screen readers.

Attribute Description
text: string Name of the color
value: string Hexadecimal of the color
